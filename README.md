# 記帳小夥伴 🐣

> 三秒記一筆，養一隻小夥伴。給我自己，也給媽媽的記帳 PWA。

## 為什麼做這個

市面上的記帳 App 一週放棄率高達七成，最大原因是「輸入太繁瑣」和「缺乏正向回饋」。
這個 App 把記帳做成 LINE 聊天室：打「午餐 120」就記好一筆，小夥伴立刻回你一句鼓勵。

## 功能

- **聊天式記帳**：打字「午餐 120」自動解析金額＋分類，或用數字鍵盤＋分類鈕
- **分類自動學習**：改過一次分類，下次同關鍵字自動套用
- **記帳小夥伴**：40 種角色可選、可取名，累積記帳 7 天進化到完全體，畢業搬進「夥伴之家」
- **錢去哪**：日／週／月統計、分類長條圖、含時間的紀錄清單
- **週報**：每週日晚上自動產生本週小結
- **匯出／匯入**：JSON 合併家人帳本（重複自動略過）、CSV 給 Excel
- **資料完全本地**：localStorage，不上雲、零後端、零營運成本

## 技術

單一 HTML 檔 PWA・Tailwind + daisyUI（CDN）・純 CSS 圖表・Service Worker 離線可用・GitHub Pages 部署

## 使用

手機瀏覽器打開網址 → 分享 → 加入主畫面，就像一個真的 App。
